# User Guide Tahap 1

## Login

Buka aplikasi, login memakai akun yang dibuat administrator atau superuser Django.

## Dashboard

Dashboard menampilkan ringkasan jumlah SP2D, D_K, dokumen, dan DRPP yang dapat diakses user.

## Menu Internal

- `SP2D`: daftar data mentah SP2D.
- `D_K`: daftar transaction detail.
- `Checklist`: status checklist dan dokumen.
- `DRPP`: daftar upload DRPP.
- `Paket SPM`: placeholder fitur Paket SPM.
- `Laporan`: placeholder export/laporan.

Tahap 1 belum memproses parser PDF, OCR, Upload Paket SPM penuh, atau Upload KW massal.
